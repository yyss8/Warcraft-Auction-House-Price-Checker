# Warcraft-Auction-House-Price-Checker

    compare item's AH price with cumulative price of its component items 

Demo: http://wow.scitnet.com

# Languages Involved
Front-End: Javascript/Bootstrap/Jquery/React/Redux/React-Router

Back-end: Nodejs

Database: Mongodb
